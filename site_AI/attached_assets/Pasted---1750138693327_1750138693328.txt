Промпт и ТЗ для дизайна дашборда

Цель:
Создать современный, интуитивно понятный и функциональный дашборд для управления AI-ассистентами, интеграциями, базой знаний и аналитикой.

Общие требования к дизайну:

Стиль: Минимализм, чистые линии, светлая/темная тема (режим переключения).
Цветовая палитра: Корпоративные цвета (например, синий/голубой акценты, нейтральные фоны).
Шрифты: Современные, легко читаемые (например, Inter, Roboto).
Адаптивность: Поддержка десктопов и планшетов.
Интерактивность: Анимации для кнопок, плавные переходы между разделами.
Структура дашборда:

0. Сообщения

Описание: Вкладка со списком всех переписок (сортировка по дате, каналу, ассистенту).
Элементы:
Фильтры: по дате, каналу (Telegram, WhatsApp и т.д.), статусу (новые/завершенные).
Поиск по ключевым словам.
Список диалогов: аватар клиента, имя, превью последнего сообщения, время.
При клике на диалог — открытие полной переписки с возможностью ответа (ручной режим).
1. Мои ассистенты

Описание: Управление ассистентами (создание, настройка, тестирование).
Элементы:
Список ассистентов: Карточки с именем, статусом (Вкл/Выкл), иконкой.
Кнопки действий:
"Добавить ассистента" (открывает форму с полями: имя, мастер-промт, тон, язык).
"Настройки" (в карточке ассистента):
Расписание (календарь/таймслоты).
Статус (toggle-переключатель).
Тональность (выпадающий список: дружелюбный, формальный, экспертный).
Язык общения (выпадающий список: русский, английский и др.).
Мастер-промт (текстовое поле с предзаполненными пресетами).
"Протестировать" (открывает чат-окно для проверки ассистента).
Чат-тест: Интерфейс аналогичный OpenAI Playground.
2. Интеграции

Описание: Подключение мессенджеров и CRM.
Элементы:
Карточки интеграций (Telegram, WhatsApp, Instagram, amoCRM, Bitrix24):
Статус подключения (✅/❌).
Кнопка "Настроить" (открывает форму для ввода API-ключей, вебхуков).
Инструкция по подключению (ссылка на модальное окно/гифку).
3. База знаний

Описание: Управление векторными базами данных для ассистентов.
Элементы:
Выбор ассистента (выпадающий список).
Добавление файлов: Drag & Drop зона для PDF/DOCX/TXT + кнопка "Google Docs".
Список документов: Название, тип, дата загрузки, кнопка "Удалить".
Статус обработки: Прогресс-бар для новых файлов.
4. Статистика

Описание: Визуализация метрик и аналитика.
Элементы:
Дашборд:
Карточки с цифрами: "Новые диалоги (за день)", "Вопросы для БД", "Активные пользователи".
Графики:
Активность по каналам (столбчатая диаграмма).
Популярные вопросы (word cloud или топ-5 списком).
Кнопка "Аналитика" (развернутый отчёт с фильтрами).
Выгрузка: Кнопка "CSV" (экспорт данных).
Фильтры: По дате, каналу, ассистенту.
5. Учетная запись

Описание: Настройки профиля пользователя.
Элементы:
Форма для смены: пароля, почты, контактных данных.
Информация о текущем тарифе.
6. Тарифы

Описание: Выбор и оплата подписки.
Элементы:
3 тариф-карточки (Базовый, Про, Бизнес) с:
Ценой, списком возможностей.
Кнопкой "Выбрать" (редирект на оплату).
7. Помощь

Описание: Поддержка пользователей.
Элементы:
FAQ: Аккордеон с вопросами/ответами.
Контакты: Email, Telegram-бот для поддержки.
Дополнительные элементы:

Хедер:
Логотип, меню, переключатель темы, уведомления (иконка колокольчика).
Сайдбар:
Иконки разделов + названия, кнопка "Свернуть".
Главный экран:
Приветствие с именем пользователя, краткая сводка (например, "Сегодня 12 новых диалогов").
